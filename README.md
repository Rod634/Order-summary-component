## 🌏 How to download and run this the project
<br>

Follow this:

```bash
    # get the project
    $ git clone https://github.com/Rod634/Order-summary-component

    # acess the path of the project
    $ cd order-summary-react

    # download the dependences from the project
    $ yarn
    or
    $ npm install

    # Run the project :)
    $ yarn start
    or
    $ npm start

```
